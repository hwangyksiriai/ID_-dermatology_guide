# 신청 폼 → 구글시트 연동 가이드

`index.html` 의 신청 폼을 구글시트와 연결하는 방법입니다. (약 3분, 무료)

## 1. 구글시트 만들기
1. [sheets.new](https://sheets.new) 로 새 구글시트를 만듭니다. (신청 데이터가 쌓일 시트)
2. 시트 이름은 자유 (예: `ID피부과_신청`).

## 2. Apps Script 붙여넣기
1. 그 시트 상단 메뉴에서 **확장 프로그램 → Apps Script** 클릭.
2. 기본 코드(`function myFunction…`)를 전부 지우고, 같은 폴더의 **`apps-script.gs`** 내용을 전부 붙여넣습니다.
3. 💾 저장 (Ctrl+S).

## 3. 웹 앱으로 배포
1. 오른쪽 위 **배포 → 새 배포** 클릭.
2. 톱니바퀴(유형 선택) → **웹 앱** 선택.
3. 설정:
   - 실행 계정: **나(본인)**
   - 액세스 권한: **모든 사용자** ← 꼭 이걸로
4. **배포** 클릭 → 권한 승인(본인 구글 계정으로 허용).
5. 배포 완료 화면에 뜨는 **웹 앱 URL** 을 복사합니다.
   (형식: `https://script.google.com/macros/s/...../exec`)

## 4. 페이지에 URL 연결
1. `index.html` 을 열고 아래 줄을 찾습니다 (script 영역 상단):
   ```js
   const GAS_URL = 'PASTE_YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';
   ```
2. 따옴표 안을 복사한 웹 앱 URL로 교체합니다:
   ```js
   const GAS_URL = 'https://script.google.com/macros/s/...../exec';
   ```
3. 저장 → 끝. 이제 신청하면 구글시트에 자동으로 행이 쌓입니다.
   (시트와 `신청일시·이름·나이·인스타그램·휴대폰·이메일·2차활용동의` 헤더는 첫 신청 때 자동 생성)

## 확인 / 참고
- 배포한 웹 앱 URL을 브라우저에서 그냥 열면 `정상 동작 중` 문구가 보이면 OK.
- 코드를 수정하면 **배포 → 배포 관리 → 편집(연필) → 버전: 새 버전 → 배포** 로 갱신해야 반영됩니다.
- URL을 연결하기 전에도 폼은 동작하며(미리보기용), URL만 넣으면 실제 시트 적재가 시작됩니다.
